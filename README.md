## 📦 Data & Integration (데이터 및 연동)
- **Excel Data Source**: `enoch_calendar_data.xlsx` 파일을 통해 기본 데이터 구조를 확인할 수 있습니다.
- **Dynamic Connection**: 구글 시트 API를 통해 전 세계 사용자가 동시에 최신 절기 데이터를 공유받습니다.

## ✨ 핵심 기능 (Key Features)
- **Divine 364-Day Cycle**: 에녹 1서와 성경적 근거에 기반한 완벽한 364일 주기 렌더링.
- **Real-time Feast Sync**: 구글 시트 연동으로 유월절, 초실절 등 절기 실시간 업데이트.
- **Global Accessibility**: 10개국 언어 지원 및 자동 감지 기능 탑재.
- **SEO & Premium UX**: 검색 엔진 최적화 및 프리미엄 다크 테마 UI 제공.

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: Next.js 14 (App Router), React
- **Styling**: Tailwind CSS, HSL Curated Palettes
- **Database**: Google Sheets API (via Apps Script)
- **Deployment**: Highly compatible with Vercel, Cloudflare Pages
