## ✨ 핵심 기능 (Key Features)
- **364-Day Order**: 분기별 일수(30, 30, 31일)를 반영한 순수 에녹달력 렌더링.
- **Biblical Feasts**: 유월절, 초실절, 오순절 등 성경적 절기 자동 강조.
- **Fixed Sabbath**: 모든 토요일을 안식일로 고정하여 시각적 강조.
- **Gregorian Sync**: 날짜 클릭 시 그레고리력(양력)과 실시간 대조 및 상세 정보 제공.
- **Global Support**: 한국어 포함 10개국 언어 자동 감지 및 지원.
- **SEO Optimized**: 검색 엔진 최적화 및 `sitemap.xml`, `robots.txt` 구성 완료.

## 🛠 기술 스택 (Tech Stack)
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS (Premium Dark Theme)
- **Language**: TypeScript
- **Icons**: Lucide React
- **Data**: Google Sheets API Integration
